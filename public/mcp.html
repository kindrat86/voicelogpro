MCP Server

VoiceLogPro — MCP Server

Voice-powered construction daily log app.

One endpoint. Stateless HTTP transport. No auth required. Paste it into any MCP client:

https://voicelogpro.com/api/mcp

Install in Claude Desktop

Add to claude_desktop_config.json:

{
  "mcpServers": {
    "voicelogpro": {
      "command": "npx",
      "args": ["mcp-remote", "https://voicelogpro.com/api/mcp"]
    }
  }
}

Install in Cursor

Add to .cursor/mcp.json:

{
  "mcpServers": {
    "voicelogpro": {
      "url": "https://voicelogpro.com/api/mcp"
    }
  }
}

Inspect

npx @modelcontextprotocol/inspector https://voicelogpro.com/api/mcp

Available Tools (3)

  • create_daily_log_template — Generate a construction daily log template.
  • get_logging_requirements — Get daily log compliance requirements.
  • list_construction_faq — Search construction logging FAQ.
