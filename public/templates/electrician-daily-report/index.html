Template Fields

Every Electrician daily report should capture these data points:

  1. Project and date
  2. Circuits/panels worked on
  3. Wire pulls completed
  4. Materials used (gauge, length)
  5. Inspections passed/failed
  6. Weather and site conditions
  7. Photos of completed work
