Request for Information (RFI) Template

A template for construction RFIs to formally request clarification from architects or owners.

Format: Markdown / PDF · Free to use · Adapted for voice daily report app workflows

What's included

Header

  • Project name and number
  • RFI number and date
  • Originator
  • Required response date

Subject

  • Drawing or spec reference
  • Question summary

Details

  • Background and context
  • Specific question
  • Proposed solution (optional)

Response

  • Architect/owner answer
  • Impact on schedule or cost
  • Signature and date
